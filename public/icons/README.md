# PWA Icon Files

Replace the placeholder PNGs in this folder with your final exports, keeping these exact filenames:

- `apple-touch-icon.png` - 180x180 - used by iPhone and iPad when saved to the home screen
- `icon-192x192.png` - 192x192 - used by the web app manifest for Android and other PWAs
- `icon-512x512.png` - 512x512 - used by the web app manifest for high-resolution PWA installs

The manifest and HTML tags already point to these paths, so you only need to overwrite the files in place.

Recommended export guidance:

- Use square PNGs
- Keep the artwork centered with comfortable padding
- Avoid relying on transparency-only edges because platforms may mask or round the icon differently
